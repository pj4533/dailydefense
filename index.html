HOW TO PLAY

// WAVE TYPES
BALANCED Standard mix of garden pests. Good wave to get established.
SWARM Fast ant hordes. High speed and count. Spider towers help slow them.
SIEGE Heavy armored beetles. Slow but very tough. Mantis towers work well.
RUSH Small groups of speedy ants. Constant pressure. Spider for crowd control.
HORDE Massive waves of everything. Full map coverage required.
// TOWERS
LADYBUG $25 Fast attack, cheap. Best DPS per dollar. Great against light pests.
SPIDER $30 Slows nearby enemies. Support tower. Pairs well with any other tower.
MANTIS $50 Long range, expensive. Best against armored beetles (SIEGE waves).
// TIPS
Wave 0 is always BALANCED — a safe start
Click a placed tower to select and sell it
New wave types unlock as you progress
